Music Genre Classification

By Andrew Wray

This small web app uses a neural network to predict the genre of a song you upload. There are 10 possible predictions that can be given, which are pop, metal, disco, blues, reggae, classical, rock, hip-hop, country, and jazz.

To use it, select a music file from your computer to be fed into the genre classifier, and then click the Predict button below. This app currently supports .wav, .ogg, and .flac file types.
